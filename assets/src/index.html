Pi Theater

This Netflix-inspired streaming site runs on a Raspberry Pi and serves videos from a connected external hard drive. The front end runs on Vue and Bootstrap, while the back end utilizes the Laravel framework for PHP which offers an easy to set up login and registration system. The playback progress of any watched movies or TV shows is maintained for logged-in users. Users logged in to their account can also add or remove titles to and from their watchlist.

The search functionality leverages the lightning fast speed of Algolia using their Instant Search components for Vue. As demonstrated in this example search, Algolia even features built-in fuzzy search capabilities to return relevant results even with typos present in the search query.

Logged-in users who are marked as "admin" can access the Upload Content page. This is where new video files that have just been transferred to the attached hard drive can have their associated attributes set in the Pi Theater database, such as the title, summary, and genres. When typing in the title field, a request is sent to an IMDb API to get the search results matching the given title. When one of those results is selected from the IMDb Search dropdown field, the rest of the fields such as the summary, year released and genres are automatically set. A poster image can either be uploaded from the user's computer, or automatically set via URL from the IMDb search result. ImageMagick automatically resizes img to the correct dimensions.

Mech Strategy Game

I began developing this strategy game, tentatively titled "Mech Monger," using the Unity game engine in early 2021. In broad terms, the gameplay combines elements of two of my favorite games: FTL, and XCOM: Enemy Unknown. Gameplay elements taken from FTL include text-based story prompts, power management, units with individually destructible components, different classes of weapons (i.e. energy weapons vs ballistic weapons), and a Rogue-like story mode. These elements are combined with features of XCOM such as turn-based squad tactical combat, base management, and 3D graphics.
