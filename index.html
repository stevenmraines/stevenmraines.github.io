Java Key Padder

Previous Next

I desired a way to control the mouse and keyboard with an XBOX style USB gamepad on my gaming PC, so that I could seamlessly switch between watching Netflix from my bed and playing a video game, without the need for three separate wireless devices.

To achieve this, I wrote a Java program which parses input from a gamepad using the Lightweight Java Game Libraries. This input is recognized and then translated into mouse and keyboard commands using the Java Robot class.

The application has two different modes: keyboard mode and mouse mode. Mouse mode allows the user to move the mouse cursor with the right analogue stick of the controller. Keyboard mode opens a circular keyboard which allows the user to type with the gamepad.

Several of the buttons are also mapped with Google Chrome and Mozilla Firefox keyboard shortcuts, making it easy to open or close a tab, switch between tabs, or go forwards or backwards a page.

Responsibilities

  • Designed the GUI using Java Swing components
  • Integrated my code with the third party Lightweight Java Game Libraries
  • Designed algorithms to translate analogue stick movement into mouse cursor movement
  • Simulated mouse and keyboard input using the Java Robot package
